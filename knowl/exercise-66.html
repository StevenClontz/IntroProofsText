Exercise 4.
Prove the following statements by either contradiction or contrapositive (be sure to note which method you used).
  1. For all integers \(n\text{,}\) if \(n^2\) is odd then \(n\) is odd.
  2. For all integers \(a\text{,}\) \(b\text{,}\) and \(c\text{,}\) if \(a\) does not divide \((bc)\text{,}\) then \(a\) does not divide \(b\text{.}\)
  3. For all integers \(a\text{,}\) \(b\text{,}\) and \(c\text{,}\) if \(a\) divides \(b\) and \(a\) does not divide \(c\text{,}\) then \(a\) does not divide \((b+c)\text{.}\)
